# AI 队友审查

`ai-reality-check` 是一个 Codex Skill，用来帮助非专业用户审查 AI 给出的建议。

它的目标不是让你更依赖 AI，而是让 AI 在跨领域任务里先讲清楚、先暴露风险、先给验证方法，再推动你行动。

## 解决什么问题

当你不熟悉某个领域时，AI 的回答可能听起来很专业，但你很难判断它到底对不对。

这个 Skill 会要求 AI：

- 先用大白话解释专业内容。
- 区分事实、假设、判断和不确定性。
- 标注建议的可信度。
- 判断任务风险和审查深度。
- 给出验证方法，而不是只让你相信 AI。
- 最后只给一个最小安全下一步。

适合用在技术方案、业务判断、工具选择、研究结论、代码解释、另一个 AI 回答复核等场景。

## 怎么安装

把本仓库复制到你的 Codex Skills 目录：

```powershell
git clone https://github.com/mvpbin/ai-reality-check.git "$env:USERPROFILE\.codex\skills\ai-reality-check"
```

然后重启 Codex，或开启一个新对话，让 Codex 重新发现这个 Skill。

## 怎么调用

你可以直接这样说：

```text
使用 ai-reality-check 帮我审查这个 AI 建议，先用大白话解释，再告诉我能不能行动。
```

也可以这样说：

```text
按 AI 队友审查模式，帮我判断这个方案靠不靠谱。
```

如果你拿到另一段 AI 回答，可以这样用：

```text
按 AI 队友审查模式，帮我看看下面这段 AI 回答有没有坑。
```

## 输出风格

默认会按这个结构回答：

```text
业务结论
大白话解释
原因
风险
验证方法
下一步
技术依据（按需阅读）
```

简单问题会压缩输出，高风险问题会更严格地拆分证据和假设。
